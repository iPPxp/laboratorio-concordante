F-000 — Proceso de Canonización
Propósito
El Laboratorio Concordante no parte de axiomas asumidos. Parte de candidatos a fundamento. Todo concepto debe justificar su permanencia antes de ingresar al Canon.
Principio
Ningún concepto posee estatus canónico por definición. Todo candidato se presume reducible hasta superar el proceso formal de auditoría.
Flujo
F_i (Candidato) → Φ → AI-000 → AI-001 → AI-002 → AI-003 → AI-004 → AI-005 → ACEPTADO = P_i (Fundamento Canónico) | RECHAZADO = Hipótesis/Definición derivada.
Criterios de promoción
No puede reducirse sin pérdida explicativa.
Es necesario para la arquitectura formal.
Aumenta el rendimiento explicativo neto.
Es consistente con el Canon vigente.
Permanece abierto a futuras auditorías.
