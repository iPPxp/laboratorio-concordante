# Registro Historico

Esta carpeta conserva conversaciones completas, transcripciones y materiales de trazabilidad.

## Regla principal

Nunca se usa como superficie normal de trabajo.

El historial puede consultarse para reconstruir por que se tomo una decision, pero una conversacion no reemplaza a un documento oficial, expediente, decision o regla de Canon.

## Formato recomendado

Guardar materiales con fecha e identificador:

```text
2026-07-01_conversacion_inicial.md
2026-07-01_auditoria_b001.md
```

## Antes de usar historial

Registrar:

- que se busca
- por que no basta el estado actual
- que documento o expediente sera afectado
