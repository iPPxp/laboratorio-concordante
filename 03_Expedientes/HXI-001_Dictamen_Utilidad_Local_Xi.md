# HXI-001 - Dictamen de utilidad local de Xi

Estatus: dictamen provisional de expediente.

Fecha: 2026-07-02.

## Proposito

Dictaminar si `Xi`, entendido provisionalmente como acoplamiento entre distincion de alternativas relacionales y restriccion de transiciones, tiene utilidad local para `PSI-001` dentro de la evaluacion `HXI-001`.

Este dictamen no admite `H-Xi`, no declara `Xi` operador vigente y no modifica Canon, documentos oficiales ni `PSI-001`.

## Base del dictamen

- `HXI-001_Criterios_Evaluacion.md`.
- `HXI-001_Mapa_Preliminar_PSI-R.md`.
- `HXI-001_Matriz_Pruebas_HXI-R.md`.
- `PSI-001_Criterio_Transformacion_Experiencia_Psicologica.md`.
- `PSI-001_Ejemplos_Conceptuales_No_Clinicos.md`.

## Dictamen corto

`Xi` tiene utilidad local acotada para describir cambios de relaciones `R` cuando hay reorganizacion relacional estable.

La utilidad no basta para admitir `H-Xi`, canonizar `Xi`, declarar un operador vigente ni reconstruir `PSI-001` desde `Xi`.

## Alcance de utilidad

La utilidad local de `Xi` se limita a tres funciones conceptuales:

- distinguir alternativas relacionales cuando no basta decir que cambio el contenido;
- describir restricciones de transicion entre configuraciones relacionales;
- marcar limites donde la comparacion `R0 -> R1` ya no procede.

Estas funciones operan solo dentro de `HXI-001` como lectura provisional, no como regla general del Laboratorio.

## Resultado por prueba

| Prueba | Resultado | Dictamen |
| --- | --- | --- |
| `HXI-R-001` | `Xi` redundante | No aporta valor suficiente frente a `CRIT-PSI-TR-001`; sirve como control negativo. |
| `HXI-R-002` | `Xi` util | Aporta valor local porque distingue alternativas relacionales y restricciones de transicion en una reorganizacion estable. |
| `HXI-R-003` | `Xi` limitado | Puede nombrar restricciones incompatibles, pero debe evitar convertir desorganizacion en reorganizacion encubierta. |
| `HXI-R-004` | `Xi` limitrofe | Sirve para marcar `no comparable`, no para explicar continuidad donde `PSI-001` exige detener la comparacion. |
| `HXI-R-005` | `Xi` bloqueado | No puede operar sobre casos no auditables ni convertir material clinico o personal real en prueba valida. |

## Decision conceptual recomendada

La recomendacion del dictamen es mantener `H-Xi` como hipotesis externa no admitida y aceptar solo una utilidad local provisional de `Xi` dentro de `HXI-001`.

No se recomienda abrir admision formal de `H-Xi` todavia.

La razon es que la evidencia local muestra un caso positivo fuerte, casos de limite y casos negativos. Eso basta para continuidad acotada, no para promocion.

## Salidas tecnicas sugeridas

El dictamen sugiere conservar estas salidas tecnicas dentro de `HXI-001`:

- `redundante`: `Xi` no agrega valor frente al criterio PSI ya disponible;
- `util_acotado`: `Xi` agrega distincion relacional sin volverse operador general;
- `limitado`: `Xi` ayuda a nombrar restricciones, pero con riesgo de sobreformalizacion;
- `no_comparable`: la comparacion `R0 -> R1` debe detenerse;
- `bloqueado`: el caso queda fuera de alcance.

Estas salidas no son todavia notacion formal. Solo ordenan la continuidad del expediente.

## Riesgos si se exagera el dictamen

- Convertir un caso positivo en admision general de `H-Xi`.
- Tratar `Xi` como operador vigente sin expediente de admision.
- Reescribir `PSI-001` desde `Xi` y perder la base ya aceptada.
- Usar `Xi` para forzar continuidad en disolucion.
- Usar `Xi` sobre casos no auditables o clinicos.

## Consecuencia operativa

`HXI-001` puede continuar, pero solo como evaluacion acotada.

El siguiente paso no debe ser canonizar ni admitir `H-Xi`. La accion limpia es decidir ruta operativa de `HXI-001` tras este dictamen: pausar, continuar con notacion minima de expediente o preparar una admision formal posterior con criterios separados.

## Resultado

Dictamen favorable a utilidad local acotada de `Xi` dentro de `HXI-001`, aceptado por `HXI-001_Decision_Estatus_Dictamen_Utilidad_Local.md`.

No autoriza admision de `H-Xi`, canonizacion de `Xi`, modificacion de documentos oficiales, cierre de `PSI-001`, uso clinico ni transformaciones materiales.
