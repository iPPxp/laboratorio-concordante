# Expedientes

Esta carpeta contiene investigacion activa, congelada o cerrada.

Un expediente debe tener:

- identificador
- estatus
- proposito
- fuentes relevantes
- hipotesis afectadas
- problemas abiertos relacionados
- decisiones asociadas
- criterio de cierre

## Estatus posibles

- abierto
- cerrado
- congelado
- rechazado
- absorbido

## Plantilla minima

```md
# ID - Titulo

Estatus:

Proposito:

Fuentes:

Hipotesis:

Problemas abiertos:

Decisiones:

Criterio de cierre:
```
