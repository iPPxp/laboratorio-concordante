# MOC-001 - Modelo Operativo de Concordancia

Estatus: expediente integrador abierto.

Fecha: 2026-07-05.

Decision de apertura: `D-2026-07-05-011`.

## Apertura

Se abre `MOC-001` como frente integrador teorico-operativo, no canonico, no clinico y no regulado.

El objetivo del expediente es estabilizar una primera version auditable del Modelo Operativo de Concordancia como modelo avanzado de construccion teorica, todavia provisional.

`MOC-001` puede formalizar operadores locales, metricas ordinales, protocolos de evaluacion, simulaciones no mutantes y disenos empiricos futuros. No puede modificar Canon, documentos oficiales, Nivel C, `Documento 04`, ni transformar el repositorio por si mismo.

## Fuentes activas permitidas

- `HXI-001_Notacion_Minima_Xi-R.md`, solo para conservar `Xi_eval` como herramienta local de evaluacion.
- `HXI-001_Evaluacion_Compuerta_Admision_Formal.md`, solo por el resultado `mantener_Xi_eval`.
- `HXI-001_Mantenimiento_Local.md`, como frontera vigente: `H-Xi` no admitida y `Xi_eval` local.
- `TCS-001_Definiciones_Minimas.md`, `TCS-001_Tipologia_Fallos_Concordancia.md`, `TCS-001_Casos_Prueba.md` y `TCS-001_Maduracion_Provisional.md`, solo como teoria provisional no canonica.
- `AO-001_Puente_Confluencia_Equivalencia.md`, `AO-001_Formalizacion_Acotada_Doc04_R001_TB.md` y `R001-001_Relacion_Formal_AO.md`, solo como antecedentes locales para proyecciones, testigos y confluencia.

## Fuentes no activas

- `PSI-001`: transferido al proyecto independiente `Psicologia Concordante`; no se usa como fuente activa interna.
- Vision papers: inactivos hasta nueva decision.
- `H-B.6` y `H-B.7`: congelados como brotes alternos y originales historicos.
- `H-Xi`: hipotesis externa no admitida; no se usa como operador ni fuente normativa.

## Frontera de alcance

`MOC-001` permite:

- definir `Xi_eval` como operador local de expediente;
- definir estados metricos cualitativos del MOC;
- construir protocolo reproducible entre evaluadores;
- usar casos sinteticos no clinicos;
- implementar software no mutante de lectura y reporte;
- disenar, sin ejecutar todavia, estudios empiricos futuros.

`MOC-001` no permite:

- admitir `H-Xi`;
- canonizar `Xi` o declararlo operador general vigente;
- evaluar personas reales;
- abrir trabajo clinico, patologico, juridico, financiero o institucional regulado;
- usar psicologia local como fuente activa;
- no permite modificar Canon, documentos oficiales o Nivel C;
- no permite modificar `Documento 04`;
- no permite cerrar `P-107`;
- no permite cerrar Confluencia global o Equivalencia global de proyecciones;
- no permite autorizar transformaciones materiales.
