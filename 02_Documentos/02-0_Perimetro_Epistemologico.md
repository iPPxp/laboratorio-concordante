# 02-0 - Perimetro Epistemologico

Estatus: documento oficial semilla.

## Proposito

Este documento define que tipo de afirmaciones puede aceptar el Laboratorio y bajo que condiciones.

## Regla central

Una afirmacion debe poder ubicarse dentro de una de estas categorias:

- definicion
- hipotesis
- teorema
- algoritmo
- expediente
- decision
- deuda conceptual
- problema abierto

## Fuera de perimetro

Quedan fuera del perimetro operativo:

- afirmaciones no trazables
- analogias tratadas como demostraciones
- intuiciones sin estatus declarado
- relecturas historicas usadas como autoridad directa
- automatizaciones que no indiquen que estatus manipulan

## Estado

Pendiente de ampliacion con criterios formales de validacion cuando el Laboratorio incorpore los documentos previos completos.
