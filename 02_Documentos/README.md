# Documentos

Esta carpeta contiene documentos oficiales del Laboratorio Concordante, uno por tema.

Los documentos deben ser breves, auditables y modificables. No deben incluir conversaciones completas ni investigacion exploratoria extensa.

Documentos iniciales:

- `00_Naturaleza.md`
- `01_Canon.md`
- `02_Fundamentos_Matematicos.md`
- `02-0_Perimetro_Epistemologico.md`
- `03_Ontologia.md`
- `04_Algebra_Operacional.md`
- `C-001_Especificacion_Tecnica_Auditor.md`

## Nivel C - Especificaciones

`Nivel C - Especificaciones` queda definido localmente en `05_Estado_Proyecto/NIVEL_C_ESPECIFICACIONES.md`.

Una especificacion tecnica oficial puede vivir en `02_Documentos` despues de auditoria y decision de promocion.

Las especificaciones candidatas permanecen en `03_Expedientes` hasta que cumplan los requisitos de `NIVEL-C-001`.
